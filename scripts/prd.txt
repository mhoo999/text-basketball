# Text Basketball Battle - 제품 요구사항 문서

## 프로젝트 개요
Text Basketball Battle은 텍스트 기반의 농구 게임으로, 선수 캐릭터 생성과 전략적 게임플레이, AI 기반 해설을 결합한 독특한 스포츠 시뮬레이션 게임입니다.

## 핵심 기능

### 1. 선수 캐릭터 생성
- 사용자는 5명의 선수 캐릭터 생성
- 각 선수 정보:
  - 이름
  - 포지션
  - 선수 설명
- AI가 선수 설명을 분석하여 능력치 산출

### 2. 게임 시스템
- 1:1 자동 매칭 시스템
- 전/후반 2쿼터제
- 각 반전 시작 전 작전 지시 입력 (20~200자)
- AI 기반 문학적 경기 해설 시스템

### 3. 바이럴 요소
- SNS 공유 기능
- 리플레이 시스템
- 시즌제 운영
